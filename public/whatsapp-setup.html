📱 Configurar WhatsApp Business API

Siga os passos abaixo para configurar o bot de atendimento automático do WhatsApp.

⚠️ WhatsApp Business API não configurado
1
Comprar Chip Dedicado

Para usar a API oficial do WhatsApp Business, você precisa de um chip dedicado:

  • Opção 1: Chip pré-pago TIM/Claro/Vivo (~R$ 50)
  • Opção 2: Serviço de terceiros como Wavy, Zenvia (~R$ 100/mês)
  • Opção 3: Meta Business API (gratuito para volume baixo)

Recomendação: Comece com chip pré-pago para testes.

2
Criar Conta Meta Business

Acesse business.facebook.com e:

  1. Criar uma conta Business
  2. Adicionar WhatsApp como ferramenta
  3. Verificar número do chip
⚠️ IMPORTANTE: Use o número do chip dedicado para verificação.
3
Configurar Webhook no ESTUDIOK

URL do webhook já está pronta:

https://estudiok.com.br/api/webhooks/whatsapp

Token de verificação:

estudiok_verify_2024

No painel do Meta Business, cole essas informações na seção Webhooks.

4
Configurar Credenciais no Vercel

Após obter as credenciais do WhatsApp Business, adicione ao Vercel:

WHATSAPP_ACCESS_TOKEN = seu_token_aqui
WHATSAPP_PHONE_NUMBER_ID = seu_id_aqui

Para adicionar no Vercel:

  1. Acesse: Vercel Environment Variables
  2. Adicione as duas variáveis acima
  3. Redeploy o projeto
5
Testar o Bot

Após configurar tudo:

  1. Envie "oi" para o número do WhatsApp Business
  2. O bot deve responder com menu interativo
  3. Verifique os logs no painel admin
